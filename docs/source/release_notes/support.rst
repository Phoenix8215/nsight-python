Support

Python Support
--------------

- Python 3.10, 3.11, 3.12, 3.13

Platform Support
----------------

-  Linux (x86_64)
-  Linux (aarch64)

GPU Support
-----------

- Turing and later GPU architectures, i.e. devices with compute capability 7.5 and higher.
