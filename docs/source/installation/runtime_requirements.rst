Runtime Requirements
=====================

* **NVIDIA Nsight Compute from CUDA toolkit 12.0+**: nsight-python uses NVIDIA Nsight Compute CLI(ncu). Please make sure that ncu version 2022.4.0 or later is installed and available in the system path.
